# Ограничения по стеку

## Обязательные

- MCP-клиент использует официальный Python SDK (пакет `mcp`)
- MCP-подключения — единственное место, где допускается asyncio
- Конфигурация MCP-серверов хранится в config/mcp-servers.md
- MCP-сервер создаётся через FastMCP из пакета `mcp`
- HTTP-запросы к внешним API — только через urllib (стандартная библиотека)
- Парсинг XML — только через xml.etree.ElementTree (стандартная библиотека)
- JSON-сериализация — только через json (стандартная библиотека)
- SQLite — только через sqlite3 (стандартная библиотека)
- Telegram Bot API — только через urllib (стандартная библиотека, POST к api.telegram.org)
- Планировщик — только while True + time.sleep (не cron, не APScheduler)
- Передача данных между MCP-инструментами — JSON-строки
