# Инварианты исследовательского агента

## Обязательные

- Не более 10 ссылок на один поисковый раунд
- Ровно два поисковых раунда: первичный и один уточняющий (не более)
- Ссылки не дублируются между раундами (дедупликация по URL)
- Финальный ответ нельзя формировать без первой суммаризации
- Прогресс в Telegram — только при смене этапа (не на каждую ссылку)
- Каждый этап должен быть записан в journal до перехода к следующему
- Финальный результат отправляется ровно один раз

## Рекомендуемые

- Поисковые запросы: 2-4 на первый раунд, 2-3 на второй
- Документы обрезаются до 3000 символов при fetch
- Если scraper не смог получить страницу — пропустить, не падать
- LLM-промпты для планирования поиска не превышают 1000 символов
- Суммаризация использует не более 8 документов за раз
