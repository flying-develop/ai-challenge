# Бизнес-правила

## Обязательные

- Это образовательный проект — код должен быть понятен студентам
- Каждый модуль содержит docstring с объяснением концепции
- Никакого продакшн-кода: без логирования в файлы, без мониторинга, без Docker
- Агент отвечает на русском языке по умолчанию

## Рекомендуемые

- Каждая новая фича сопровождается примером использования
- Комментарии объясняют "зачем", а не "что"
- README обновляется при добавлении новой функциональности
