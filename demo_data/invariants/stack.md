# Ограничения по стеку

## Обязательные

- Python 3.11+ — единственный язык разработки
- SQLite — единственная база данных (никаких PostgreSQL, MongoDB, Redis)
- Подключение к LLM через HTTP API (requests/httpx), без SDK-обёрток
- Три поддерживаемых провайдера: OpenAI, Anthropic, Qwen

- MCP-клиент использует официальный Python SDK (пакет `mcp`)
- MCP-подключения — единственное место, где допускается asyncio
- Конфигурация MCP-серверов хранится в config/mcp-servers.md

## Рекомендуемые

- Предпочитать стандартную библиотеку Python внешним пакетам
- Минимум зависимостей в requirements.txt
- Тесты через pytest
