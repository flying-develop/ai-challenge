# Архитектурные инварианты

## Обязательные

- Проект — CLI-приложение, никаких веб-интерфейсов, API-серверов или TUI-фреймворков
- Архитектура — один процесс, синхронный Python (никакого asyncio для основного цикла)
- Конфигурация хранится в MD-файлах в директории config/
- История диалогов и таски хранятся в SQLite

## Рекомендуемые

- Модули не более 200 строк
- Функции не более 30 строк
- Плоская структура импортов (без циклических зависимостей)
- MCP-клиент оборачивает asyncio в синхронные методы
